About This Site

FlashDrop Pro is a browser-based peer-to-peer file and text sharing website. It allows users to transfer data directly between devices without uploading to cloud servers.

The site uses WebRTC to establish secure, temporary connections between browsers. When optional password protection is enabled, files and messages are encrypted locally using AES-256 before transmission.

No accounts are required. No files are stored. No content is logged. Once a session ends or the page is closed, all transfer data is automatically discarded.

How FlashDrop Pro Works

  • Direct browser-to-browser connections
  • Optional AES-256 client-side encryption
  • 6-digit code, link, or QR-based pairing
  • Temporary, self-destructing sessions
  • No server-side file access

This site is designed so that only you and your recipient can access shared content — even the service itself cannot read encrypted data.

关于本站

FlashDrop Pro 是一个基于浏览器的点对点文件与文本传输网站，允许用户在设备之间直接传输数据，而无需上传至云服务器。

本站使用 WebRTC 技术在浏览器之间建立安全、临时的连接。当启用可选密码保护时，文件和文本会在发送端浏览器中使用 AES-256 加密后再进行传输。

无需注册账号，不存储文件，不记录内容。会话结束或页面关闭后，所有传输数据都会自动销毁。

FlashDrop Pro 的工作方式

  • 浏览器之间直接连接
  • 可选的 AES-256 客户端加密
  • 6 位验证码 / 链接 / 二维码配对
  • 临时、自毁式传输会话
  • 服务器无法访问文件内容

本站的设计目标是：只有发送者和接收者才能访问内容，甚至平台本身也无法读取加密数据。
