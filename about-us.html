About Us

Why FlashDrop Pro Exists

FlashDrop Pro was created around a simple idea: most file-sharing tools do not need access to your data — yet many are built to collect it.

Accounts, cloud uploads, and tracking have become defaults. We believe privacy should be the default instead.

Our Philosophy

If we don’t need your data, we should never touch it.

  • No file storage
  • No activity logs
  • No user profiling
  • Optional client-side encryption

Design Choices

When optional passwords are enabled, content is encrypted locally using AES-256 before transfer. This ensures only the intended recipient can access the data.

关于我们

为什么创建 FlashDrop Pro

FlashDrop Pro 源于一个简单的想法：大多数文件共享工具并不需要访问你的数据，但它们却往往这样做。

账户体系、云端存储、用户追踪成为默认，我们认为隐私才应该是默认。

我们的理念

如果我们不需要你的数据，就永远不应该接触它。

  • 不存储文件
  • 不记录日志
  • 不建立用户画像
  • 可选的客户端加密

设计选择

当启用可选密码时，内容会在浏览器本地使用 AES-256 加密后再进行传输，只有接收方可以解密。

私たちについて

存在理由

FlashDrop Pro は「本来必要のないデータまで取得してしまう共有ツール」の問題から生まれました。

理念

必要のないデータには、決して触れない。

  • ファイル保存なし
  • ログなし

  • 任意のクライアント暗号化

設計思想

회사 소개

존재 이유

FlashDrop Pro는 필요하지 않은 데이터까지 수집하는 파일 공유 방식에 대한 문제의식에서 시작되었습니다.

철학

필요 없는 데이터는 절대 다루지 않는다.

  • 파일 저장 없음
  • 로그 기록 없음
  • 사용자 프로필 없음
  • 선택적 클라이언트 암호화

설계 원칙

비밀번호 사용 시 AES-256 암호화가 브라우저에서 적용되어 수신자만 접근할 수 있습니다.
